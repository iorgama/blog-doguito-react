## Descrição do projeto:

Projeto desenvolvido durante a formação de React Js da Alura. O objetivo foi desenvolver um blog para listar notícias relacionadas aos pets.

## Bibliotecas e tecnologias utilizadas:

- React Router Dom
- Axios
- React Js
